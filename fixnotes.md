То есть надо переименовать поля на фронте: заметки тренера => общие заметки (А внутри поля подсказка "эту запись будет видеть ваш тренер")

Личные заметки - переводить не надо, но в подсказке внутри поля нужно вставить "Эта запись видна только вам".
